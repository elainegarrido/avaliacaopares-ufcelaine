Avaliação por Pares entre Equipes

Formulário web para equipes de até 10 integrantes, com coleta de e-mail, avaliação individual por critérios e apuração da nota individual pela moda das notas finais atribuídas pelos pares.

Orientações

Cada estudante deve preencher seus dados, informar a equipe e avaliar separadamente cada colega do grupo. A nota de cada colega em uma avaliação é a média dos 4 critérios. Ao final, a nota individual do aluno é calculada pela moda dessas notas recebidas dos pares. Em caso de empate na moda, o sistema adota a maior moda.
Este arquivo já funciona para coleta local no navegador e geração de relatório. Para coleta real de vários respondentes, publique a página e conecte um endpoint de armazenamento (por exemplo: Google Apps Script + Google Sheets, Supabase ou Firebase). O campo de configuração está no código como SCRIPT_ENDPOINT.

Dados do avaliador

Preferencialmente e-mail institucional. Também aceita Gmail, Outlook, Hotmail ou Live.
O sistema abrirá automaticamente o número correto de colegas a serem avaliados: total da equipe menos você.

Critérios de avaliação

Escala de notas por critério: 1 a 5.
1 = não atendeu · 2 = atendeu pouco · 3 = atendeu parcialmente · 4 = atendeu bem · 5 = atendeu plenamente

Apuração da nota individual

Regras de cálculo usadas nesta página:
  1. Para cada aluno avaliado em uma resposta, calcula-se a média dos 4 critérios.
  2. A média é convertida para nota com uma casa decimal.
  3. A nota individual final do aluno é a moda das notas finais recebidas dos pares.
  4. Se houver empate de frequência, o sistema adota a maior moda.
O relatório lê as respostas salvas localmente no navegador. Se você conectar um endpoint externo, mantenha também um relatório em planilha ou banco de dados.
Configuração opcional para coleta online: abra o arquivo e substitua a constante SCRIPT_ENDPOINT pela URL de um Web App do Google Apps Script ou outro endpoint que receba POST em JSON. Esta página já envia um objeto estruturado com dados do avaliador, equipe, timestamp e avaliações dos colegas.
